# Product Overview

VPS Deployer is a web-based tool for automated deployment of PHP and Laravel applications to VPS servers.

## Core Functionality

- One-click deployment of Docker-based PHP/Laravel environments
- SSH-based remote execution on target VPS servers
- Automatic Traefik reverse proxy setup with Let's Encrypt SSL
- MySQL database provisioning with auto-generated credentials
- Real-time deployment logs streamed to the browser

## Target Users

Developers who need to quickly provision PHP/Laravel environments on VPS servers without manual Docker/Nginx configuration.

## Key Features

- PHP 7.3 and 8.3 support
- Laravel or pure PHP project types
- Optional custom domain with automatic HTTPS
- Force overwrite option for redeployments
- Credential file generation for deployed projects
